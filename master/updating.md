# Updating

::: tip 
We recommend following the post-update deployment procedures on the [Deployment](deployment.html) page.
:::


## Update via Composer

```bash
$ cd /home/pixelfed # Or wherever you chose to install web applications
$ composer update
```

## Update via Git

```bash
$ cd /home/pixelfed # Or wherever you chose to install web applications
$ git pull origin dev
```
